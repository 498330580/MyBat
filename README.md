# 🦇 MyBat 
📁 用来存放一些实用的Windows批处理脚本

---

## 📜 脚本列表及功能说明

### 🎯 split_mp3.bat
🎵 **功能**: 自动将当前目录下的MP3文件平均分割成两部分  
🔧 **依赖**: 需要安装ffmpeg并添加到系统PATH环境变量  
📂 **输出文件**: 对每个输入文件`filename.mp3`会生成:  
  - `filename_part1.mp3` (前半部分)  
  - `filename_part2.mp3` (后半部分)  
🛠️ **使用方法**:  
  1. 将MP3文件放入脚本所在目录  
  2. 双击运行脚本  
  3. 分割完成后按任意键退出  

---

💡 提示：所有脚本均为Windows批处理文件(.bat)，双击即可运行
